- 修复 Svg 无法显示 的问题

## 修改内容：
- 将页面内头像改为本地头像
- 将原本的背景替换为动态樱花背景（参考自 [fwxiaohan.github.io](https://github.com/fwxiaohan/fwxiaohan.github.io "fwxiaohan.github.io") ）
- 去除`<li>`标签的锚点链接，改为文本链接
- 去除超链接下方的横线
- 去除页面中间的黑色块块
- 增加显示访问者IP地址（调用 https://pv.sohu.com/cityjson?ie=utf-8 接口）
